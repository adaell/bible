# bible
Looks up a bible verse from biblegateway.com and then
prints it to the console. Uses the ESV by default.

Usage:

```
bible [book] [chapter] [verse (optional)] [translation (optional)]
```

Example usage:

```
bible Genesis 2:7-8
bible Genesis 2 7-8
bible Exodus 4 ESV
bible Exodus 4 3
bible 1 John 3 10-22
bible 1Samuel 2 3-5 KJV
```
